# Clarrity Desktop Environment

Clarrity is a lightweight and modern desktop environment inspired by LXDE, but rebuilt with GTK3/GTK4 and Wayland support.  
It aims to preserve the simplicity and speed of classic lightweight desktops while embracing modern technologies.  

## ⚠️ Development Status
Clarrity is **not yet developed**.  
This repository currently serves as a placeholder for future source code, documentation, and community contributions.  

## ✨ Planned Features
- **Waybox**: A Wayland compositor and window manager, forked from Openbox.  
- **Maine**: A GTK4-based file manager, forked from PCManFM.  
- **CLPanel**: Lightweight panel for tasks and menus.  
- **CLSession**: Session manager adapted for Wayland.  
- **CLAppearance**: Theme and appearance configuration tool.  
- **CLTerminal**: Lightweight terminal emulator.  
- **CLPower**: Power management utilities.  
- **CDM (Clarrity Display Manager)**: A fork of LXDM, modernized with GTK3/GTK4.  

## 📦 Distribution Roadmap
- Alpha: Debian, Devuan  
- Pre-Beta: Ubuntu  
- Beta: Fedora, RHEL, Fedora-based distros  
- Beta Advanced: Arch, Manjaro  
- Final: Gentoo  

## 🔧 Goals
- Ultra-lightweight performance for old and modern hardware.  
- Full Wayland integration with XWayland fallback.  
- Modern GTK3/GTK4 toolkit support.  
- Clear branding with CL-prefixed applications.  
- Open development and community-driven contributions.
